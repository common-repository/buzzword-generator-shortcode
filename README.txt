== Description ==

* Makes a random phrase of marketing buzzwords anywhere you insert the `[makebullshit]` shortcode. Just insert the `[makebullshit]` short code anywhere you'd like magic marketing words to appear.

* It uses three arrays; the first contains a verb, the second contains an adjective, and the third contains a noun.

== Installation ==

* Install and activate. Use the [makebullshit] short code anywhere you'd like to display the text.
